## Usage with Docker

Just run:

```
make build
```

Then go to `http://localhost:8030/health-check` to check all is ok.

## Usage from local

First of all you should execute 

```
make prepare-local
```

And then start local environment:

```
make start-local
```

And then going to `http://localhost:8030/health-check` to check all is ok.
